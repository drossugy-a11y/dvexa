# 项目需求文档

Stock Radar — AI 量化选股系统需求定义（已确认版）。

## 核心目标

构建一个 **AI Agent 驱动的半自动中线量化选股系统**：

1. **用户策略驱动** — 用户定义选股策略逻辑，Agent 执行
2. **AI 研究分析** — Agent 对策略筛选出的股票做深度研究
3. **中线持仓** — 持仓周期 1-4 周
4. **半自动流程** — Agent 选股 → 人工确认 → 手动下单
5. **研究报告** — Agent 自动生成研究报告

```
用户定义策略
     ↓

     ↓
┌────────────────────────────────────────┐

│       ↓                                │
│  AI 研究分析 → 生成研究报告             │
│  (LLM分析)    (入选原因/逻辑/风险)      │
└────────────────────────────────────────┘
     ↓
展示候选股票 + 研究报告
     ↓
人工确认 → 手动下单
```

## 用户工作流

```
1. 用户在系统中定义/调整选股策略
2. 用户点击"开始分析"触发 Agent
3. Agent 自动执行：
   - 从 akshare 获取 A 股数据
   - 计算五维因子评分
   - 按策略规则筛选候选股票
   - 对每只候选股票做 AI 研究分析
   - 自动生成研究报告
4. 系统展示 Top N 候选 + 研究报告
5. 用户阅读报告，人工决策
6. 用户手动在券商下单
```

## 因子设计（自行设计）

### 基本面因子（五维评分）

| 因子 | 计算方式 | 权重 |
|------|----------|------|
| 成长性 | 营收增长率、净利润增长率、趋势 | 25% |
| 盈利能力 | ROE、毛利率、持续性 | 25% |
| 估值 | PE/PB 区间评分、历史分位 | 20% |
| 财务健康 | 资产负债率、流动比率、现金流 | 15% |
| 质量 | 毛利率稳定性、ROE 持续性 | 15% |

### 技术面因子（自行设计）

中线选股重点关注的技术因子：

| 因子 | 设计思路 | 中线意义 |
|------|----------|----------|
| **趋势因子** | 价格与均线关系（MA20/MA60） | 确认中期趋势方向 |
| **动量因子** | N日收益率、相对强弱 | 衡量上涨动能 |
| **量价因子** | 成交量变化率、量价配合 | 验证趋势有效性 |
| **波动因子** | 历史波动率、ATR | 评估风险水平 |
| **突破因子** | 价格突破关键位、新高 | 捕捉启动信号 |

### 因子计算示例

```python
# factor.py — 自行设计的因子计算

def trend_factor(prices, period=20):
    """趋势因子：价格相对均线位置"""
    ma = prices.rolling(period).mean()
    return (prices - ma) / ma * 100

def momentum_factor(prices, period=20):
    """动量因子：N日收益率"""
    return prices.pct_change(period) * 100

def volume_factor(volumes, period=5):
    """量价因子：成交量相对均量"""
    ma_vol = volumes.rolling(period).mean()
    return volumes / ma_vol

def breakout_factor(prices, period=60):
    """突破因子：是否突破N日新高"""
    high = prices.rolling(period).max()
    return (prices >= high).astype(int)
```

## 市场状态判断与策略切换

### 三种市场状态

| 状态 | 判断依据 | 策略风格 |
|------|----------|----------|
| **牛市** | 指数站上MA60 + MACD金叉 + 量能放大 | 进攻型：追涨强势股 |
| **熊市** | 指数跌破MA60 + MACD死叉 + 量能萎缩 | 防守型：超跌反弹或空仓 |
| **震荡市** | 指数在MA60附近震荡 + 无明显趋势 | 均衡型：选有支撑的优质股 |

### 市场状态判断因子

```python
# market_regime.py — 市场状态判断

def judge_market_regime(index_data):
    """
    根据大盘指数判断市场状态
    输入：沪深300或上证指数历史数据
    输出：'bull' / 'bear' / 'shock'
    """
    close = index_data['close']
    
    # 均线系统
    ma20 = close.rolling(20).mean()
    ma60 = close.rolling(60).mean()
    
    # MACD
    ema12 = close.ewm(span=12).mean()
    ema26 = close.ewm(span=26).mean()
    dif = ema12 - ema26
    dea = dif.ewm(span=9).mean()
    
    # 判断逻辑
    if close.iloc[-1] > ma60.iloc[-1] and dif.iloc[-1] > dea.iloc[-1]:
        return 'bull'  # 牛市
    elif close.iloc[-1] < ma60.iloc[-1] and dif.iloc[-1] < dea.iloc[-1]:
        return 'bear'  # 熊市
    else:
        return 'shock' # 震荡市
```

## Agent 研究报告（已确认：标准版）

每只候选股票生成标准版研究报告（约2页）：

```markdown
## [股票名称] 研究报告
**日期：** 2024-01-15  |  **评分：** 85/100  |  **建议：** 关注

---

### 一、入选原因
- **基本面**：ROE 18%，连续3年稳定增长，毛利率42%行业领先
- **技术面**：突破20日均线，MACD金叉，量能放大1.5倍
- **资金面**：北向资金连续3日净流入，机构持仓增加

### 二、中线逻辑
- **行业景气**：下游需求旺盛，订单可见度覆盖未来2个季度
- **估值安全**：当前PE 28倍，处于历史30%分位，有安全边际
- **动量确认**：技术形态突破，动量因子评分92，趋势向上

### 三、风险提示
- ⚠️ 大盘系统性风险：若指数回调可能带动个股调整
- ⚠️ 行业政策变动：需关注监管政策变化
- ⚠️ 业绩不及预期：Q1财报若低于预期可能引发回调

### 四、观察建议
- **建议**：关注，可在回调至20日均线附近介入
- **目标仓位**：15%（单只上限）
- **止损位**：跌破20日均线或亏损8%
- **预期周期**：2-3周
```

报告特点：
- 结构化四段式（入选原因 → 中线逻辑 → 风险提示 → 观察建议）
- 数据驱动（引用具体指标数值）
- 中线视角（关注中期趋势而非短期波动）
- 风险前置（明确标注风险因素）
