# 项目技术栈

Stock Radar 项目使用的技术栈清单。

## 后端 (stock_radar/)

| 技术 | 版本 | 用途 |
|------|------|------|
| Python | 3.10+ | 运行时 |
| FastAPI | 0.110+ | REST API 框架 |
| uvicorn | 0.27+ | ASGI 服务器 |
| akshare | 1.14+ | A股数据获取 |
| pandas | 2.0+ | 数据处理 |
| SQLite | 内置 | 本地缓存 |
| OpenAI API | 1.0+ | LLM 解释 |
| plotly | 5.18+ | 可视化 |
| python-dotenv | 1.0+ | 配置管理 |

## 前端 (DENG-main/)

| 技术 | 版本 | 用途 |
|------|------|------|
| React | 19.0 | UI 框架 |
| TypeScript | 5.8 | 类型安全 |
| Vite | 6.2 | 构建工具 |
| Tailwind CSS | 4.1 | 样式框架 |
| Lucide React | 0.546 | 图标库 |
| Motion | 12.23 | 动画库 |

## 开发工具

| 工具 | 用途 |
|------|------|
| Claude Code | AI 辅助开发 |
| Playwright MCP | 浏览器自动化测试 |
| Obsidian | 知识库管理 |
| Git | 版本控制 |

## 启动命令

```bash
# 后端
cd stock_radar
uvicorn api:app --port 8000

# 前端
cd DENG-main
npm run dev

# 一键启动
双击 start.bat
```
